# 功能說明

## 🌍 語言切換

在導航欄右上角，你會看到一個帶有語言圖標的按鈕，顯示當前語言（ZH 或 EN）。

**使用方法：**

1. 點擊語言按鈕（顯示 "ZH" 或 "EN"）
2. 語言會在中文和英文之間切換
3. 設定會自動保存到瀏覽器

**功能特色：**

- 即時切換，無需重新載入頁面
- 自動保存語言偏好設定
- 所有界面文字和工具都會切換語言

## 🌙 深色模式

在語言切換按鈕旁邊，你會看到一個月亮或太陽圖標的按鈕。

**使用方法：**

1. 點擊深色模式按鈕（月亮圖標 = 切換到深色模式，太陽圖標 = 切換到淺色模式）
2. 整個應用程式會立即切換主題
3. 設定會自動保存到瀏覽器

**功能特色：**

- 即時切換主題
- 自動保存主題偏好設定
- 所有工具和組件都支援深色模式
- 護眼的深色配色方案

## 📱 PWA 功能

當你在支援的瀏覽器中訪問應用程式時，可能會看到安裝提示。

**安裝方法：**

1. 在 Chrome、Edge 或 Safari 中開啟應用程式
2. 等待安裝提示出現（通常在頁面底部）
3. 點擊「安裝」按鈕
4. 應用程式會被安裝到你的設備上

**PWA 優勢：**

- 離線使用：即使沒有網路連線也能使用工具
- 原生體驗：像原生應用程式一樣運行
- 自動更新：新版本會自動更新
- 快速啟動：從主畫面直接啟動

## 🔧 工具使用

**開啟工具：**

1. 在主頁面點擊任何工具卡片
2. 工具會在彈出視窗中開啟
3. 點擊視窗外部或右上角的 X 按鈕關閉

**搜尋工具：**

1. 使用頂部的搜尋框
2. 輸入工具名稱、描述或標籤
3. 結果會即時過濾顯示

## 📱 手機使用

**響應式設計：**

- 所有功能在手機上都能正常使用
- 工具視窗在手機上會全螢幕顯示
- 觸控友好的按鈕和界面

**手機安裝：**

1. 在手機瀏覽器中開啟應用程式
2. 等待「加入主畫面」提示
3. 按照提示安裝到主畫面
4. 像原生應用程式一樣使用

## 🛠️ 疑難排解

**語言切換不工作：**

- 確保 JavaScript 已啟用
- 嘗試重新整理頁面
- 檢查瀏覽器控制台是否有錯誤

**深色模式不工作：**

- 確保瀏覽器支援 CSS 自定義屬性
- 嘗試清除瀏覽器快取
- 檢查是否有瀏覽器擴充功能干擾

**PWA 安裝問題：**

- 確保使用 HTTPS 連線
- 確保瀏覽器支援 PWA（Chrome、Edge、Safari）
- 檢查是否已經安裝過應用程式

**工具無法開啟：**

- 確保 JavaScript 已啟用
- 嘗試重新整理頁面
- 檢查瀏覽器控制台錯誤訊息

## 💡 提示

- 所有設定都會自動保存到瀏覽器
- 應用程式支援離線使用
- 定期檢查更新以獲得最新功能
- 在手機上安裝 PWA 以獲得最佳體驗
